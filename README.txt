Allows you to import the feed of a public page or profile on your website as post or custom post type.

== Description ==

With Facebookpress you can import your page status, photos, albums, events and videos.
The plugin gives the following options :
1 - Facebook page ID finder.
2 - Post Type Map, gives you option to save the imported feeds on specific post type and specific taxonomy.
3 - Import images as post thumbnail.
4 - Import images into the post content (as album).
5 - The facebook feeds imported as "pending" post status.
Notes about facebook feed types:
- Some of the feeds doesn't has name or it has generic name like "Timeline Photos, PAGENAME cover photo or Photos from".
- If a feed doesn't has name or has some of the generic words, the plugin will generate a title from the feed message by taking first 6 words.
- If the feed type is album, the images will be in the WP post content and the first one will be the post thumbnail.
- Status and photo will be consider as post.
- Video feed will be imported as ifram video (hosted by source).
